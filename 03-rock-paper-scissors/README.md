# Rock, Paper, Scissors (Project 3)

A simple Python implementation of the classic Rock, Paper, Scissors game played against the computer.

## How to run

From the repo root:

```bash
python3 03-rock-paper-scissors/rock_paper_scissors.py
```

What I practiced

- while True loops
- input validation with in
- lists and indexing
- random choice generation
- tracking game state with counters
