# Number Guessing Game (Project 2)

A Python game where the user selects a range and tries to guess a randomly generated number.

## How to run

From the repo root:

```bash
python3 02-number-guessing-game/number_guessing_game.py
```
What I practiced

-input validation using .isdigit()

-converting strings to integers

-while True loops

-using the random module

-tracking attempts with a counter
